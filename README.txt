l33t_gray2 is a port of Tara Chen's l33t_gray for B2evolution Skins 1.x to Skins 2.0 and slightly rewritten to accommodate most of the new Skins features namely the widgets.

The file is now very minimal as most of the funky bits like post lists and calendar stuff is now handled by B2evolution's widgets.
